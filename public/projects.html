SamePath

Student Collaboration Platform

SamePath is a comprehensive mobile application designed to help students connect with classmates, form study groups, and collaborate on academic projects. The platform integrates with university class schedules (including Virginia Tech) to automatically suggest study partners based on shared classes and availability.

Key Features:

  • Class schedule integration with CRN lookup
  • Automatic study partner matching based on shared classes
  • Group chat functionality for teams
  • Free time discovery and coordination
  • Friends list management
  • User preferences and notifications
  • Contact matching and acceptance flow

Technologies Used:

React Native TypeScript Python Flask MongoDB Express.js REST API

SportsPredictor

ML-Powered Sports Analytics

An advanced sports prediction system that uses machine learning algorithms to forecast game outcomes for NFL, MLB, and College Football. The system factors in team statistics, player injuries, weather conditions, and historical performance data to generate accurate predictions.

Key Features:

  • Machine learning-based predictions for multiple sports
  • Real-time injury tracking and impact analysis
  • Weather condition integration for outdoor sports
  • Automated data scraping and processing
  • Park factors and pitcher statistics (MLB)
  • Batch prediction generation for weekly games
  • Historical data analysis and trending

Technologies Used:

Python Machine Learning Web Scraping Pandas NumPy scikit-learn
