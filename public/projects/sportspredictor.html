Data Pipeline

SportsPredictor

An NFL analytics tool that scrapes structured data, cleans it nightly, and ships matchup predictions with explainable metrics.

Role
Developer · Data Pipeline Engineer
Timeline
Fall 2023 – Present
Stack
Python · Pandas · scikit-learn · BeautifulSoup
Output
Weekly matchup reports + interactive CLI

What the build delivers

SportsPredictor keeps weekly NFL data organized so matchup forecasts stay transparent and easy to share.

  • Nightly ETL routine that refreshes league stats, injuries, weather, and betting lines before predictions ship.
  • Feature engineering and scikit-learn classifiers output confidence percentages plus the metrics that drove the pick.
  • CLI tooling for both batch reports and single-matchup lookups, making it simple to check a game on demand.

My focus

  • Designed the data schema and scrapers to keep sources normalized and debuggable.
  • Tuned model training pipelines and validation so updates stay current without overfitting.
  • Built the reporting layer that explains every pick in plain language.
