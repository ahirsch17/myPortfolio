Data Pipeline

SportsPredictor

Python pipeline that refreshes NFL data nightly and ships explainable matchup forecasts.

SportsPredictor normalizes league stats, injuries, weather, and odds so every forecast includes confidence percentages and the metrics that drove the pick.

Run it yourself

Follow these steps to boot the API and open the dashboard locally.

  1. Download and unzip sportsPredictor.zip.
  2. Open a terminal in the unzipped folder and run:
    cd sportsPredictor && python -m uvicorn api.server:app --reload --port 8001
  3. In your file explorer, open sportsPredictor/web/index.html to view the UI.

What it delivers

Keep weekly predictions fast, transparent, and easy to share with friends following the games.

Automated ETL

Scrapers normalize stats, weather, injuries, and betting lines so inputs stay consistent before each run.

Explainable models

scikit-learn classifiers output confidence ranges plus the metrics that tipped the matchup.

CLI tooling

Batch scripts and single-game flags make it simple to check a weekly slate or a single matchup on demand.

Inside the build

Modular architecture keeps the nightly refresh under five minutes and the forecasts easy to validate.

Data ingestion

Requests layer handles rate limiting and retries for public APIs, with lightweight caching to avoid duplicate pulls.

Feature engineering

Transforms raw stats into model-ready features like pressure rates, red-zone efficiency, and injury impact scores.

Reporting

Formatted CLI outputs highlight the forecast, top drivers, and a short summary you can paste straight into group chats.
