Data Pipeline

SportsPredictor

An NFL analytics tool that scrapes structured data, cleans it nightly, and ships matchup predictions with explainable metrics.

Role
Developer · Data Pipeline Engineer
Timeline
Fall 2023 – Present
Stack
Python · Pandas · scikit-learn · BeautifulSoup
Output
Weekly matchup reports + interactive CLI

From raw feeds to explainable forecasts

The project started as a CLI experiment to check weather-adjusted spreads. It evolved into a modular pipeline that scrapes injury reports, refreshes team stats, trains scikit-learn models, and ships human-readable summaries that explain why a pick is favored.

Pipeline at a glance

  • Daily scrapers normalize NFL data from public league sources, weather services, and injury reports.
  • Feature engineering transforms drive efficiency, injury impact, and surface conditions into model-ready features.
  • scikit-learn classifiers tuned for NFL matchups output confidence percentages instead of binary picks.
  • Report generator highlights key drivers like pressure rates, secondary matchups, and red-zone efficiency.

Engineering notes

The ETL layer focuses on transparency and maintainability. Batched requests and caching keep nightly runs fast, and models are refreshed weekly with rolling windows so predictions stay current without overfitting.

CLI for power users

Analysts can jump into a terminal to run single-game predictions, export CSVs, and tweak weighting schemes for conservative, neutral, or upset-friendly outputs.

Data products

Artifacts from the pipeline—CLI outputs, batch scripts, and data refresh dashboards.

Impact & next steps

  • Pipelines now run nightly to keep stats, injuries, and weather aligned before prediction day.
  • CLI tooling makes single-matchup checks quick for friends following weekly games.
  • Summaries highlight the main drivers behind a pick so the reasoning is easy to share.

Roadmap

Next steps include a hosted web UI with authentication, integrating live odds streaming, and turning the summaries into printable one-pagers.
