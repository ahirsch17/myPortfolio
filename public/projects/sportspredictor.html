Data Pipeline

SportsPredictor

A cross-sport analytics engine that scrapes structured data, cleans it nightly, and ships matchup predictions with explainable metrics for NFL, MLB, and College Football fans.

Role
ML Engineer · Data Pipeline Developer
Timeline
Fall 2023 – Present
Stack
Python · Pandas · scikit-learn · BeautifulSoup
Output
Weekly matchup reports + interactive CLI

From raw feeds to explainable forecasts

The project started as a CLI experiment to check weather-adjusted spreads. It evolved into a modular pipeline that scrapes injury reports, pulls park factors, trains models, and ships human-readable PDFs that explain why a pick is favored.

Pipeline at a glance

  • Daily scrapers normalize data from league APIs, weather services, and betting lines.
  • Feature engineering transforms historical stats, injury impact, and surface effects into model-ready tensors.
  • Gradient boosting and ensemble models per sport output probability distributions, not just binary picks.
  • Report generator highlights key drivers: QB pressure rates, pitcher fatigue, or home-park inflation.

Engineering notes

I focused on performance and transparency. The ETL layer uses async requests and caching to keep nightly runs under five minutes. Models are re-trained weekly with rolling windows to prevent overfitting while staying current. Explanations are generated with SHAP values and simplified to bullet points for the final PDF.

CLI for power users

Analysts can jump into a terminal to run single-game predictions, view feature importance charts, or export CSVs for their own parlay spreadsheets. Options flags toggle between conservative, neutral, and upset-friendly weighting schemes.

Data products

Artifacts from the pipeline—batch scripts, reports, and injury dashboards.

Impact & next steps

  • Early users reported a 14% improvement in pick accuracy when using matchup explanations.
  • Pipeline uptime has stayed above 99% since migrating to async fetchers.
  • Reports now feed a Notion database that updates friends’ private betting dashboards.

Roadmap

I’m exploring a hosted web UI with authentication, integrating live odds streaming, and fine-tuning prompt-based summaries to speed up weekly report drafting.
